# Book Recommender System

## Create a conda virtual environment
Please run following commands in order in your terminal:
```bash
  $ conda create --prefix ./env python==3.9 -y
```
```bash
  $ conda activate ./env/
```
```bash
  $ pip install -r requirement.txt
```
